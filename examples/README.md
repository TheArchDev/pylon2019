
## Get started

    pip install pgzero

    pgzrun game.py


Builtins for:

- Screen
- Rect / Actor
- images, sounds and music
- Events: mouse, keyboard
- Animations
- Clock


## Docs

https://pygame-zero.readthedocs.io


### To install the latest dev version:

    pip install -e git+git@github.com:lordmauve/pgzero.git#egg=pgzero[repl]


This allows:

    pgzrun --repl game.py
